# NPC 字典 (NPCs)

游戏"勇者大陆"中所有 NPC 的完整列表。talk_to_npc 类型条件引用的 npc_id 必须来自此列表。任务的 npc_id（发布者）也必须是此列表中存在的 NPC。

## npc_elder - 村长老莫

- **npc_id**: npc_elder
- **名称**: 村长老莫
- **角色**: 村庄长老，负责新手引导和主线任务推进
- **性格**: 慈祥稳重，说话简洁，喜欢用谚语教导年轻人
- **位置**: x=500, y=300
- **可发布任务类型**: kill_monster（打怪任务）
- **精灵图**: npc_elder

## npc_blacksmith - 铁匠格雷

- **npc_id**: npc_blacksmith
- **名称**: 铁匠格雷
- **角色**: 武器店铁匠，指导玩家装备强化和材料收集
- **性格**: 粗犷豪爽，热情直率，对武器和装备充满热忱
- **位置**: x=350, y=450
- **可发布任务类型**: collect_item（道具收集任务，收集材料、装备等）
- **精灵图**: npc_blacksmith

## npc_merchant - 商人莉娜

- **npc_id**: npc_merchant
- **名称**: 商人莉娜
- **角色**: 流浪商人，提供道具交易相关任务和情报线索
- **性格**: 精明狡黠，话语间夹带商业推销，但本性善良
- **位置**: x=650, y=400
- **可发布任务类型**: use_item（使用道具任务，引导玩家使用背包中的装备或道具）
- **精灵图**: npc_merchant

## npc_healer - 药师艾琳

- **npc_id**: npc_healer
- **名称**: 药师艾琳
- **角色**: 教堂药师，引导玩家使用药水和恢复类道具
- **性格**: 温柔细心，关心玩家的健康状态，说话轻声细语
- **位置**: x=480, y=250
- **可发布任务类型**: use_item（使用药水/恢复类任务）
- **精灵图**: npc_healer
